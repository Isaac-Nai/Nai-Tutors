# Nai Tutors

Nai Tutors is a free tutoring platform designed to make academic support more accessible for students. The goal is to connect learners with resources, guidance, and (eventually) tutors in a simple, intuitive, and modern web experience.

This project is currently in active development as a full-stack web application.

---

## 🚀 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- React
- Next.js
- Tailwind CSS

Backend:
- Node.js
- Express (planned / in progress)
- PostgreSQL

Deployment:
- Vercel
- GitHub

---

## 🎯 Project Goals

- Build a clean and accessible tutoring platform
- Allow students to find academic help easily
- Enable tutors to create profiles and share resources
- Learn and demonstrate modern full-stack development skills
- Create a strong portfolio project for internships and placements

---

## 🧠 Features (Planned / In Progress)

### Core Features
- User authentication (student / tutor roles)
- Tutor profile pages
- Subject-based browsing system
- Resource sharing (notes, links, etc.)
- Basic dashboard for users

### Future Features
- Booking system for tutoring sessions
- Messaging between students and tutors
- Ratings and reviews
- AI-assisted study helper
- Email notifications
